Assembly Cheatsheet

x86-64 (64-bit) Calling Convention

Arguments: 1st: RDI, 2nd: RSI, 3rd: RDX, 4th: RCX, 5th: R8, 6th: R9

Caller-saved: RAX, RCX, RDX, RDI, RSI, R8-R11

Callee-saved: RBX, RBP, R12-R15

Return Value: RAX

x86 (32-bit) Calling Conventions

cdecl (C default)
- Args: Stack (Right-to-Left)
- Stack Cleanup: Caller
stdcall (WinAPI)
- Args: Stack (Right-to-Left)
- Stack Cleanup: Callee (using ret N)
fastcall
- Args: 1st: ECX, 2nd: EDX, Rest: Stack
- Stack Cleanup: Callee

Caller-saved (x86): EAX, ECX, EDX   |   Return Value: EAX
